>JSP 파일 업로드 기능 구현 순서

1 데이터베이스 구축하기
2 업로드 양식 페이지 작성하기
3 데이터베이스 연동 클래스 작성하기
4 업로드 처리 페이지 작성하기
5 파일 다운로드 페이지 작성하기(파일을 다운로드한 횟수 DB에 저장하기 포함)



데이터베이스 구축하기(MySQL)

	> 파일 업로드에서 반드시 구현되어야 할 정보 두가지
		첫번째. 서버에 저장된 실제 파일의 이름
		두번째. 사용자가 저장한 파일의 이름

	> 구현하는 방법 2가지
		첫번째. 게시판, 프로필 등의 테이블에 파일 관련 속성을 삽입하기
		두번째. 따로 파일 관련 테이블 생성하기 


	> 테이블 구축하기

		/*실습용 데이터베이스 생성*/
		CREATE DATABASE FILE;
		USE FILE;

		/*파일 업로드 테이블 생성 */
		CREATE TABLE FILE(
			FILENAME VARCHAR(200),    /*업로드할 파일의 원본이름*/
			FILEREALNAME VARCHAR(200) /*실제 업로드된 파일이름*/
		);


파일을 다운로드 한 횟수 저장하기
	
	>테이블에 새로운 속성을 추가하기

		/*FILE테이블에 새로운 속성 downloadCount 추가*/
		ALTER TABLE FILE ADD (downloadCount INT);
